# Petrol Price Analysis & Macroeconomic Impact

## 📌 Overview
This project analyzes how the Russia-Ukraine conflict impacted petrol prices and GDP across 13 countries using data science techniques.

## 🧠 Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Seaborn & Matplotlib

## ⚙️ Pipeline
1. Data Collection  
2. Data Cleaning  
3. Feature Engineering  
4. Encoding (Label, One-Hot, Target, Frequency)  
5. Scaling (Min-Max, Standardization)  
6. Exploratory Data Analysis  
7. Visualization  
8. Modeling  
9. Evaluation  
10. Insights  

## 📊 Key Insights
- Higher oil dependency → greater GDP decline  
- South Asia & Middle East most affected  
- North America showed resilience
- Pakistan had highest price increase
