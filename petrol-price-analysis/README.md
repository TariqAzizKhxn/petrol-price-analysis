# Petrol Price Analysis & Macroeconomic Impact

## 📌 Overview
This project analyzes the impact of the Russia-Ukraine conflict on petrol prices and GDP across 13 countries using data science techniques.

## 🧠 Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Seaborn & Matplotlib

## ⚙️ Workflow
1. Data Cleaning  
2. Feature Engineering  
3. Encoding Techniques  
4. Scaling & Transformations  
5. Data Visualization  
6. Insight Extraction  

## 📊 Key Insights
- High oil dependency leads to higher GDP decline  
- South Asia & Middle East were most affected  
- North America showed resilience  
- Pakistan saw the highest price increase
